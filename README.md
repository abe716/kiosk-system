# Kiosk Machine System

A Python-based kiosk application that allows users to browse items, place orders, and view totals.

## Features
- Interactive menu system
- Item selection and ordering functionality
- Calculates total cost of selected items
- Handles user input and navigation

## Technologies
- Python

## Purpose
This project demonstrates the development of an interactive system using structured programming and object-oriented principles.

## How to Run
1. Download the project files
2. Open in a Python environment
3. Run the main program file
